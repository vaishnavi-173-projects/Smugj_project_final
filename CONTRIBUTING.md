# Contributing to Smugj

First off, thank you for considering contributing to Smugj! It's people like you that make Smugj such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check if there's already an [issue](https://github.com/vaishnavi-173-projects/Smugj_project_final/issues) open for it. If not, open a new issue.

## Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints.
5. Issue that pull request!

## Setting up for local development

1. Clone the repository
2. Install Python dependencies: `pip install -r requirements.txt`
3. Install Terraform CLI
4. Setup Azure CLI and authenticate

## Coding Style

- Terraform: Run `terraform fmt` before committing.
- Python: Follow PEP 8 guidelines.
- Markdown: Keep it clean and readable.

Thank you for contributing!
